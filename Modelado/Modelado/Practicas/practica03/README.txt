Para compilar el archivo.c se debe de meter el comando en la terminal:
        gcc quicksort.c -o quicksort -fopenmp
luego para correr el archivo se debe de meter el comando en la terminal:
        ./quicksort
y al iniciar el programa te pedira que metas el tamaño del arreglo que
quieres ordenar (se debe de meter un numero entero para que funcione,
si se mete cualquier otro tipo de dato no funcionara) y al final te
regresara un arreglo ordenado del tamaño que metiste.
